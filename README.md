# 3xui-shopbot | Telegram-бот для продажи VPN

**3xui-shopbot** — комплексное решение для автоматизированной продажи VPN-ключей через Telegram с интеграцией в панель **3x-ui** и веб-панелью управления.

## 🚀 Возможности

### Основная функциональность
- ✅ Полная автоматизация: от регистрации до выдачи ключа после оплаты
- ✅ Мульти-хост: управление несколькими серверами 3x-ui из одной панели
- ✅ Гибкие тарифы: настройка сроков и цены для каждого хоста
- ✅ Тестовый период (trial) с настраиваемой длительностью

### Платежные системы
- 💳 **YooKassa** — СБП и банковские карты
- 🤖 **CryptoBot** — криптовалюта (USDT, BTC, TON, ETH и др.)
- 💎 **Heleket** — карты и криптовалюта
- 🪙 **TON Connect** — оплата через TON Wallet
- ⭐ **Telegram Stars** — внутренняя валюта Telegram

### Маркетинг и вовлечение
- 📢 Принудительная подписка на канал (опционально)
- 🤝 Реферальная система (3 типа начислений):
  - Процент с покупки реферала
  - Фиксированная сумма за покупку
  - Бонус пригласившему при старте
- 💼 Оплата с внутреннего баланса

### Поддержка и мониторинг
- 🆘 Support-бот для обращений пользователей
- 📊 Тест скорости по хостам (SSH speedtest)
- 📈 Веб-панель с полной статистикой
- 💾 Автоматический бэкап базы данных

### Соответствие требованиям
- 📄 Условия использования (встроенные)
- 🔒 Политика конфиденциальности (встроенная)
- ✅ Принятие документов пользователем

## 📋 Требования

1. Сервер Ubuntu/Debian с доступом по SSH
2. Домен с A-записью на IP сервера
3. Панель 3x-ui на одном или нескольких серверах

## 🛠️ Установка

```bash
curl -sSL https://raw.githubusercontent.com/HolyLittleGirl/3xui-shopbot/main/install.sh | sudo bash
```

### В процессе установки:
1. Введите домен (например, `shop.example.com`)
2. Укажите email для SSL (Let's Encrypt)
3. Дождитесь завершения

### Доступ к панели:
```
URL: https://your-domain.com/login
Логин: admin
Пароль: admin
```

> ⚠️ **Смените пароль после первого входа!**

## ⚙️ Настройка

### 1. Telegram бот
**Настройки → Основные:**
- Токен бота (от [@BotFather](https://t.me/BotFather))
- Username бота (без @)
- Telegram ID администратора (через [@userinfobot](https://t.me/userinfobot))

### 2. Хосты 3x-ui
**Настройки → Управление хостами → Добавить хост:**
- Название (произвольное)
- URL панели 3x-ui
- Логин/пароль от 3x-ui
- Inbound ID (обычно 1 или 2)
- Ссылка на подписку (для этого хоста)

### 3. Тарифы
**Настройки → Управление хостами → Добавить тариф:**
- Название (например, "1 месяц")
- Срок (в месяцах)
- Цена в RUB
- Цена в Telegram Stars (опционально)

### 4. Платежные системы
**Настройки → Платежи:**

| Система | Что нужно |
|---------|-----------|
| YooKassa | Shop ID + Secret Key |
| YooMoney | Wallet ID + API Key |
| CryptoBot | API токен из @CryptoBot |
| Heleket | Merchant ID + API Key |
| TON Connect | Кошелек + TON API Key |

### 5. Запуск бота
**Кнопка "Запустить бота" в шапке панели**

Или включите **Режим Production** для автозапуска:
**Настройки → Настройки панели → 🚀 Режим Production**

## 📊 Дополнительные настройки

### Speedtest
**Настройки → Настройки панели → Speedtest (каждые N минут)**
- По умолчанию: 60 минут
- Рекомендуется: 60-120 минут
- Минимум: 5 минут

### Автобэкап БД
**Настройки → Настройки панели → Автобэкап базы (каждые N дней)**
- По умолчанию: 1 день
- 0 — отключить

### Реферальная система
**Настройки → Рефералы:**
- Тип начисления (процент / фикс / бонус при старте)
- Процент / сумма
- Скидка приглашённому
- Минимальная сумма на вывод

### Поддержка
**Настройки → Support-бот:**
- Токен support-бота
- Username support-бота
- ID forum-чата (опционально)

## 💾 Бэкап и восстановление

### Бэкап через панель
**Настройки → Настройки панели → Скачать бэкап БД**

### Ручной бэкап
```bash
docker cp 3xui-shopbot:/app/project/data/users.db ./users.db.backup
```

### Восстановление
```bash
docker-compose down
docker cp ./users.db.backup 3xui-shopbot:/app/project/data/users.db
docker-compose up -d
```

> ⚠️ **Не удаляйте Docker volume `shopbot-db`!**
